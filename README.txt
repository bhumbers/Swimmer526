Running the GUI:
"java -jar SwimGUI.jar"

Running the optimizer (which will save the results to controlData/CHARID.csv")
"java -jar SwimOptimization.jar CHARID", 
where CHARID is the ID of a character. Character names used in final paper:
paddle
tadpole
humanFlyRefTraj

See PDF paper writeup for details on this project
